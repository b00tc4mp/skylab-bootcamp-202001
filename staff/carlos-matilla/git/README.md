# git


## git init

initializes a repo
```sh
$ git init
```

## git status

shows the status of the repo

```sh
$git status
```

## git add

add content and changes to be tracked

```sh
$git add staff
```

### git add .

adding all files

## git commit

creates a history point

```sh
$ git commit -m "add readme"
```

## git log

shows the commit's history and branches

```sh
git log
```

### git log --graph

shows history in a vertical graph mode

```sh
git log --graph
```

## git config

showing the config
```sh
git config
```

## git diff

showing the differences with the copy
