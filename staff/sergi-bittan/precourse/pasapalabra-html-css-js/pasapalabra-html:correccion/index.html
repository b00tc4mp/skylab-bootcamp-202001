Instrucciones

Se le van a efectuar una serie de preguntas siguiendo en orden alfabético. Por cada acierto el rosco se irá completando de color verde y por cada error de color rojo.
En caso de no conocer o no estar seguro de la respuesta puede escribir PASAPALABRA y dejaría la pregunta para la siguiente ronda, en caso de llegar a ella.
Dispone de un tiempo máximo de 5 min para completar el rosco. El número mayor de aciertos conseguidos antes de agotar el tiempo será la puntuación obtenida.
