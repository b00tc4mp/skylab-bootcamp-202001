# Command-Line Interfacte (CLI)

Most common used commands

## [pwd]()

- list files

```sh
$ ls
```

- list hidden files

```sh
$ ls -a
```

- list files with details

```sh
$ ls -l
```

- create directory

```sh
$ mkdir
```

- remove file

```sh
$ rm
```

- remove directory
```sh
$ rmdir
```

- change directory

```sh
$ cd [dir-name]
```

- father directory

```sh
$ cd ..
```

- create file

```sh
$ touch
```

- copy a file 

```sh
$cp [original-file][new-file]
```

- create link

```sh
$ ln -s [path]
```

- open text editor

```sh
$ nano
```
