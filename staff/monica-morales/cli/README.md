# Command-Line Interface(CLI)

Most common used commands.
![alt CLI](https://media.giphy.com/media/26BoEH5DpMz6bw05q/source.gif)
## [pwd](https://en.wikipedia.org/wiki/Pwd)
- list files
```sh
$ ls
```
- list hidden files
```sh
$ ls -a
```
- list files with details
```sh
$ ls -l
```
- list with details and human-readable sizes
```sh
$ ls -h
```
- create folders
```sh
$ mkdir
```
- edit files
```sh
$ nano
```
- see last edit files
```sh
$ cat
```
- change permissions
```sh
$ chmod
```
- recursive
````sh
$ -r
````
- rename
````sh
$ mv
````
- move files
````sh
$ mv
````
- delete files
````sh
$ rm
````
- copy files
````sh
$ cp
````
- create files
````sh
$ touch
````
- clear screen
````sh
$ clear
````
- historial
````sh
$ history
````
